# 📚 Offline Student AI Assistant 🤖

## 🔍 Overview

This project is an offline AI assistant for students. It uses Streamlit and Ollama to provide simple and clear answers.

## 🚀 Features

- Chat interface
- Runs offline using Ollama
- Easy explanations for students

## 🛠️ Tech Stack

- Python
- Streamlit
- Ollama

## ▶️ How to Run

1. Start Ollama:
   brew services start ollama

2. Run the app:
   streamlit run app.py
   
## 🧠 Model Setup

This project uses TinyLlama via Ollama.

### Steps:
1. Install Ollama: https://ollama.com  
2. Run the model:
   ollama run tinyllama  

Note: Model file is not included due to large size.
